# List-Tuple-Dictionary

## List

儲存有順序的資料，由[]包起來，組成元素由逗號隔開

### 建立方法
```
list1 = []
list2 = list()
list3 = ['chinese', 'english']
list4 = list('hello')           # ['h', 'e', 'l', 'l', 'o']
```

### 取得元素
```
list1 = ['chinese', 'english', 'japanese']
list1[0]    #chinese
list1[1]    #english
list1[-1]   #japanese, -1從結尾回來
```
